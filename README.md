## Features
*   **Instant Streaming**: No waiting for downloads. Stream torrents directly.
*   **Universal Playback**: Real-time transcoding engine supports MKV, AC3, DTS, and more.
*   **Clean UI**: No ads, no tracking, just content.
*   **Cross-Platform**: Built on Electron and React.
